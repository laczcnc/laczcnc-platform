-------------------
- Cotizaciones y pedidos aparecen como listas compactas.
- Estados editables directamente desde la vista previa.

- Talleres muestra primero la lista y deja el formulario bajo “Nuevo taller”.
- Entregas muestra seguimiento de todos los pedidos.
- Productos incluye vista previa de la imagen principal.
- Galería admite imágenes y videos y usa una vista pública compacta.
- Tienda incluye carrito; el formulario de cotización se abre bajo demanda.
- Usuarios aparece como lista compacta y el formulario de edición se despliega.
- El enlace redundante “Crear usuario” se elimina del menú lateral.
- Administradores pueden asignar acceso individual a secciones del panel.
- Contacto y pie de página leen la configuración pública del negocio.
- Buscadores públicos ocupan menos espacio y se abren cuando se necesitan.


ACCESOS INDIVIDUALES
--------------------
La migración agrega profiles.section_access como JSON.

- Una sección con valor true queda habilitada.
- Una sección con valor false queda bloqueada.
- Una sección ausente conserva el acceso normal determinado por el rol.
- La cuenta con rol admin mantiene acceso total.


CONTRASEÑAS
-----------
Supabase Auth no permite recuperar ni mostrar contraseñas existentes, ni
siquiera al administrador. La aplicación permite crear credenciales y
reemplazar una contraseña, pero nunca leer la contraseña actual.

NEXT_PUBLIC_SUPABASE_ANON_KEY
SUPABASE_SECRET_KEY

No incluyas estas claves dentro del código ni del repositorio.
